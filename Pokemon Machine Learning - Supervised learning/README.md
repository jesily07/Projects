# Pokemon Dataset Machine Learning

Pokemon is a series of video games developed by Game Freak and published by Nintendo and The Pokemon Company. The core games are released in generations, each with different Pokemon, storylines, and characters. 

Dataset consists of Player's acheievements and performance in different categories. We will conduct EDA on Player's performance with Total pokemons collected and test Machine Learning models.

## Libraries

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Algorithms

- Random Forest
